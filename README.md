# bahoot
A game-based learning platform like kahoot but opensource!

## Full Stack

- HTML5
- CSS (Tailwind CSS)
- JavaScript (TypeScript)
- ReactJS (NextJS)
- Express (Node.js)
- PostgreSQL
- MongoDB
- CI/CD with Github workflow
- Docker (Devops)

## Backend

- Express
- Prisma

## Frontend

- Nextjs

## Sofeware version

- Node: v20.10.0
